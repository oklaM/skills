# 预期输出模板

## 任务执行输出

```
### TASK-XXX - 测试任务名称
- **描述**: 具体描述...
- **产出**: tests/report.md
- **状态**: ✅ 完成
```

## 反思报告输出

```
### YYYY-MM-DD（第 N 次）
- **目标推进**: 简述目标推进情况
- **完成任务**: TASK-XXX, TASK-YYY
- **阻塞问题**: 无 / 具体问题描述
- **下次优先**: 下一阶段任务
```

## 测试报告输出

```
## 测试报告 - YYYY-MM-DD HH:MM

### 执行结果

| 测试ID | 测试名称 | 结果 | 备注 |
|--------|----------|------|------|
| T001 | 目标读取测试 | ✅ PASS | |

### 统计
- 总计: 8
- 通过: 8
- 失败: 0
- 跳过: 0
```
